## X 项目总结

1.在整个项目的搭建过程中，最重要也是最耗费时间的过程便是设计过程，包括但不限于数据库设计，数据结构设计。最开始我没有考虑到数据库管理的合理性，导致数据库连接对象的管理存在问题，后来在深入学习设计模式的过程将，将数据库管理调整为了单例设计模式。至于数据库表该如何设计也是一大难题，如果考虑牛津高阶这种详解词典来设计单词数据总表，显然会大大增加开发难度，
